# Projeto
O tema central das fontes fornecidas é a investigação histórica, militar e sociológica da incidência de Objetos Voadores Não Identificados (OVNIs) no Brasil, com ênfase absoluta na Operação Prato, uma missão militar sigilosa deflagrada pela Força Aérea Brasileira (FAB) entre 1977 e 1978 na região amazônica.
